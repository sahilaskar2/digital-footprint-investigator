# Digital Footprint Investigation and Profiling Tool

This project is a web application developed for the Cyber Forensic and Investigation course.

It helps investigators analyze a person's digital footprint by checking their online presence across multiple platforms.

Features
- Username investigation
- Digital footprint profiling
- Online presence analysis
- Investigation report generation

Technologies Used
- Python
- Flask
- HTML/CSS
- GitHub

Developed for:
Cyber Forensic and Investigation (PECCS604T)
